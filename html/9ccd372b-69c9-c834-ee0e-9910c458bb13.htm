Remarks
Definition from ISO/CD 10303-43:1992: A representation is one or more representation items that are related in a specified representation context as the representation of some concept. Definition from IAI: The IfcRepresentation defines the general concept of representing product properties. NOTE The definition of this entity relates to the STEP entity representation. Please refer to ISO/IS 10303-43:1994 for the final definition of the formal standard. HISTORY New entity in IFC Release 2.0 IFC2x Edition 3 NOTE Users should not instantiate the entity IfcRepresentation from IFC2x Edition 3 onwards. It will be changed into an ABSTRACT supertype in future releases of IFC. IFC2x Edition 3 CHANGE The inverse attributes LayerAssignments and RepresentationMap have been added with upward compatibility.
