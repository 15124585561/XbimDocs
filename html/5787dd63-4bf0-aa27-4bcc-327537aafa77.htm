Remarks
Definition from IAI: The physical quantity, IfcPhysicalQuantity, is an abstract entity that holds a complex or simple quantity measure together with a semantic definition of the usage for the single or several measure value. The Name attribute defines the actual usage or kind of measure. The interpretation of the name label has to be established within the actual exchange context. In addition an informative text may be associated to each quantity by the Description attribute. HISTORY New entity in IFC Release 2.x. It replaces the calcXxx attributes used in previous IFC Releases.
