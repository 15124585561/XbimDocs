Remarks
Definition from IAI: The IfcPropertyReferenceValue allows a property value to be given by referencing other entities within the resource definitions of IFC. Those other entities are regarded as predefined complex properties and can be aggregated within a property set (IfcPropertySet). The allowable entities to be used as value references are given by the IfcObjectReferenceSelect. HISTORY: New entity in IFC Release 1.5. Entity has been renamed from IfcObjectReference in IFC Release 2x.
